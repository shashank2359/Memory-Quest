# 🐾 Memory Quest

A premium kids-themed Memory Card Matching Game built using React and Vite.

## Features

* 🎮 Interactive memory card game
* 🐻 Cute animal-themed cards
* 🎯 Multiple difficulty levels
* ⏱ Timer starts on first card click
* 📊 Move counter
* 🏆 Best score tracking using Local Storage
* ✨ Card flip animations
* 🎉 Win celebration effects
* 🔊 Sound effects
* 📱 Fully responsive design

## Tech Stack

* React
* Vite
* JavaScript
* CSS3
* Local Storage

## Installation

```bash
npm install
npm run dev
```

## Deployment

Deployed using Vercel.
